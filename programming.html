Programming Examples

Example 1

Unit Conversion

You can visit the site and use the unit converter by clicking here!

The Unit Conversion program was a lot of fun to work on to me simply because I enjoy coding. The assignments being AJAX assignments made it fun as I got to code the design as well as the program. When it comes to Javascript Programs, I always started with the front-end development to get the look of the program down before starting the program aspect. A main issue I had when working on this was getting the same number returned that I used for my input. All that told me was that the program wasn't working and so I had to learn to inspect the site and debug it to see exactly where the problem was occurring. The problem stemmed from my not having linked the unit conversion file from my Professors site so the program could know what a mile was, a centimeter, an inch, etc. Once I plugged this in, I had little to know problem working out the rest of the tweaks on the program.
